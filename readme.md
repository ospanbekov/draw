## Requirements

- Laravel `5.8` or newer
- PHP `7.1` or newer

## Installation

`git clone https://github.com/ospanbekov/draw.git`

`composer install`

`php artisan key:generate`

`php artisan migrate:fresh --seed`

`npm install && npm run dev`
